# Calculator-App-Python
Calculator developed in Python with a user interface.

Python Calculator (GUI)
> Overview

This project is a desktop calculator application developed in Python using the Tkinter library.
It simulates a modern mobile calculator interface, focusing on usability, clean design, and real-world application of programming concepts.

The application goes beyond basic console interaction by providing a graphical user interface (GUI), handling user inputs through buttons and keyboard events.

- Features
- Basic operations (Addition, Subtraction, Multiplication, Division)
- Advanced operations:
   Square (x²)
   Square root (√x)
- Interactive graphical interface (GUI)
- Keyboard input support
- Error handling (invalid expressions)
- Mobile-inspired layout design
- Interface Preview

> A clean and intuitive layout inspired by smartphone calculators, built entirely with Tkinter.


> Technologies Used
-Python 3
-Tkinter (GUI)
-Event-driven programming
-Object-Oriented Programming (OOP)
